To launch this interactive Python3 program, follow the below steps:

1. For Microsoft Windows OS and Apple macOS, install Python from www.python.org. For Linux OS, use sudo apt install python3.

2. Open Terminal, type and enter: pip install numpy requests pandas scipy sympy matplotlib django pymongo.

3. Navigate to the directory you downloaded the file to.
 
4. For Windows OS, launch by typing: python main.py / For macOS and Linux, launch by typing: python3 main.py

5. Enjoy the interactive CLI program. A new version will be released soon with the input-output features.

6. This program is free of any malicious or hidden code and software. Feel free to launch this program.
